# ၂၀၂၆ ခုနှစ် မြန်မာ့ပို့ကုန်သွင်းကုန်လိုင်စင်ဆိုင်ရာ စည်းမျဉ်းအသစ်များ အကျဉ်းချုပ်

၂၀၂၆ ခုနှစ်၊ ဇန်နဝါရီလ ၁ ရက်နေ့မှစတင်၍ မြန်မာနိုင်ငံ၏ ပို့ကုန်သွင်းကုန်လိုင်စင်ဆိုင်ရာ စည်းမျဉ်းအသစ်များကို အောက်ပါအတိုင်း ပြောင်းလဲသတ်မှတ်ခဲ့ပါသည်။ ဤစည်းမျဉ်းအသစ်များကို ကုန်သွယ်ရေးဦးစီးဌာနမှ အမိန့်ကြော်ငြာစာအမှတ် ၄/၂၀၂၅ ဖြင့် ထုတ်ပြန်ခဲ့ခြင်းဖြစ်သည်။

## အဓိကအချက်များ

*   **သက်တမ်း:** ပို့ကုန်/သွင်းကုန်လိုင်စင်နှင့် ပါမစ်များ၏ သက်တမ်းကို ရက်ပေါင်း ၁၈၀ အဖြစ် သတ်မှတ်ထားပါသည်။ [1, 2, 3]
*   **လျှောက်ထားမှု ကန့်သတ်ချက်:** ကုန်သွယ်မှုရည်ရွယ်ချက်ဖြင့် သွင်းကုန်များအတွက် ကုမ္ပဏီတစ်ခုလျှင် တစ်လအတွင်း ကုန်ပစ္စည်းအမျိုးအစား (HS Code တူညီသော) တစ်ခုအတွက် လိုင်စင်လျှောက်ထားမှု တစ်ကြိမ်သာ ခွင့်ပြုမည်ဖြစ်သည်။ [1, 2, 3]
*   **အွန်လိုင်းစနစ်:** လိုင်စင်လျှောက်ထားမှုအားလုံးကို Myanmar TradeNet 2.0 အွန်လိုင်းစနစ်မှတစ်ဆင့်သာ တင်သွင်းပြီး ဆောင်ရွက်ရမည်ဖြစ်သည်။ [4]
